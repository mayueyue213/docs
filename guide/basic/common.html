# 下划线

下划线 _ ，明确用来表现一些”内部“，”私有“，”局部“ 的含义，如内部文件，内部变量。

文件，文件夹，URL的命名，尽量采用中划线 -。

使用 _ 需要有明确的理由

# 大写字母

大写往往只在驼峰，类命名，常量等处用到，大写字母开头的场景较少

使用 大写字母 开头 需要有明确的理由
