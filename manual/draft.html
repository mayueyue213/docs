# 函数表达式和函数声明

首先来区分一下，什么是函数表达式，什么是函数声明，有一个很简单的判断方法: 函数表达式可以省略函数名，函数声明不行。

// 函数表达式
setTimeout(function foo() {
  // ...
}, 1000);

// 函数表达式
(function foo() {
  // ...
})();

// 函数声明
function foo() {
  // ...
}
