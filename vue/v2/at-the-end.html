# 写在最后

用了很久的Vue 2，在Vue 3已经发布，大家已经开始研究并应用的2021年，才来写这系列的文章， 有点“过时”的感觉，不得不承认的是体现了对这些核心知识的掌握十分滞后。

我个人长期处于会用，并且算不得精通使用的程度，断断续续看了一些源码， 从最终的效果来说，对结构，对原理有一定的印象和一些粗浅的理解，但基本都是一看就会，一问就懵的状态。

对于Vue 2的研究的文章多如牛毛，而且在视角，知识的关联上各不相同，我也看过一些其他人的学习总结，但是别人的理解并不是个人的理解，别人消化后的二手经验很多难以下咽。只有尝试去产出，才能在总结的过程中形成知识的串联。在学习和应用Vue 3的过程中，同时再对Vue 2做个系统的学习总结，就是期望能尝试和改善学习方法，在总结的过程中，做到有效学习有效产出，检验掌握程度，同时对Vue 2和Vue 3的区别，对比形成更清晰的认知。

很多的技术人都在学习和应用技术的时候，做到及时，可持续并且有效地深入理解和产出，这是难能可贵的。 Vue 2的应用还将持续较长的时间，而Vue 3也逐步进入到实际应用阶段，这个时间深入研究Vue 2并不“过时”，只是太过滞后，但没及时有效掌握是事实，痛定思痛，承认以往的庸碌也并没有那么难。

及时，可持续并且有效地深入理解和产出，在学习和工作的同时，就能够利用所学所用知识去构造影响力，这是互联网技术行业的优势，很遗憾，工作的前三年，因为庸碌和懒惰，错失了过多的机会。痛定思痛，痛何如哉，行动，迈出第一步，坚持，持续有效学习，有效产出，长期的坚持，构建个人品牌及影响力。2021年02月25日凌晨于北京朝阳区
