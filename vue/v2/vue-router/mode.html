# mode

简要说明各种模式的区别

abstract 模式就是用于 server 端的，它由于没有window.history，没有用户的前进后退等，因此逻辑上很简单，就是在不同的页面之间进行切换，只需要一个堆栈就能保持这些结构，核心是说，它需要保存不同路由下的记录，但是不需要进行 内部的router.history 和 浏览器的window.history的同步。
