# AST

AST 全名 Abstract Syntax Tree ，中文叫做 抽象语法树，AST 就是对源码抽象的结构化表示。在需要对源码进行解析的地方，AST 往往都是必不可少的一个环节。

# AST Element

所谓的 AST 元素，其实没听起来那么神奇，就是一个描述对象

function createASTElement (
  tag: string,
  attrs: Array<ASTAttr>,
  parent: ASTElement | void
): ASTElement {
  return {
    type: 1,
    tag,
    attrsList: attrs,
    attrsMap: makeAttrsMap(attrs),
    rawAttrsMap: {},
    parent,
    children: []
  }
}
